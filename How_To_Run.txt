How To Run app.py:
 
1. Create a new environment for the project
(Command:- conda create -n burnout)

2. Activate the Environment
(Command:- conda activate burnout)

3. Install all the requirements given in requirements.txt
(Command:- pip install -r requirements.txt)

4. Now run the command :  python app.py

5. Paste the IP Address on a Chrome Browser , for the animation and text to work and display properly.
